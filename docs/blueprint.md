# **App Name**: Shamshop Clone

## Core Features:

- Product Display: Display products with images and details, mirroring the original website's product presentation, sizes and responsiveness.
- Category Navigation: Implement the same category and subcategory structure as the original website for easy browsing.
- Image Handling: Correct sizing for product images. Responsive carousel of images at banner size
- Mobile Responsive Design: Replicate the mobile-friendly design of the original website for a seamless user experience on all devices.
- Shopping Cart: Implement a shopping cart feature to allow users to add and manage products they wish to purchase.

## Style Guidelines:

- Primary color: Soft, muted beige (#F5F5DC) to create a sense of calm and sophistication.
- Background color: Very light beige (#FAFAF0) for a clean, airy feel.
- Accent color: Pale salmon pink (#FA8072) for highlighting important elements and CTAs, offering a subtle contrast.
- Body and headline font: 'Inter', a sans-serif, chosen for its modern and clean appearance suitable for both headlines and body text.
- Use simple, elegant icons in a dark gray color (#333333) to maintain a consistent visual style.
- Replicate the exact layout of the original website, ensuring that image sizes, banner sizes, and product placements are consistent.
- Incorporate subtle animations like fades or transitions for a smooth user experience.
